## Setup:

If you use npm:

    $ npm install global b-games

If you use yarn:

    $ yarn global add b-games

## Usage:

The package contains four text games now. To start them you should type 
     
    $ brain-games
Or    
    
    $ brain-even
    
Or

    $ brain-calc

Or

    $ brain-gcd

Enjoy!
