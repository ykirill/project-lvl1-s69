## Setup:

If you use npm:

    $ npm install global b-games

If you use yarn:

    $ yarn global add b-games

## Usage:

The package contains seven text games:

- brain-games
- brain-even
- brain-calc
- brain-gcd
- brain-balance
- brain-progression
- brain-prime

To start one of them just type it name in your terminal.

Enjoy!
